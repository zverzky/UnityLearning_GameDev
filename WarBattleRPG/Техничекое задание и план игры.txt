План игры:

Общее описание
	Игра представляет собой пошаговую многопользовательскую RPG с боями персонажей и прокачкой. 
	Игроки выбирают классы персонажей, сражаются с нейтральными мобами для получения опыта и улучшают свои навыки.

		Ключевые механики

		1)Выбор класса персонажа: Игроки могут выбрать из нескольких классов (например, воин, маг, стрелок),
		каждый из которых имеет уникальные способности.
		2)Бой: Игроки и мобы сражаются в пошаговом режиме, используя специальные атаки и предметы.
		3)Прокачка уровней: После убийства мобов игроки получают опыт, что позволяет им повышать уровень, 
		увеличивать основные характеристики и открывать новые способности.
		4)Способности: Каждому классу доступны уникальные навыки, которые можно улучшать.
		5)Мир игры: Игроки могут исследовать карту, находить разных мобов и получать ресурсы.

		Уровень разработки

		1)Графика: Использование 2D или 3D графики в зависимости от вашего видения и возможностей.
		2)Sound Design: Музыкальное оформление и звуковые эффекты для создания атмосферы.
		3)Пользовательский интерфейс (UI): Меню выбора класса, интерфейс боя, инвентарь и экран уровня.
Техническое задание:

Структура приложения

	1)Изучить архитектуру Unity для интеграции с C#.
	2)Создать основные классы, такие как Player, Mob, Ability, GameManager, UIManager.

	3)Классы персонажей
	Создайте базовый класс Character с общими атрибутами (здоровье, уровень, опыт).
	Создайте производные классы для каждого типа персонажа с уникальными способностями.

	4)Система боев
	Реализовать анимации боя.
	Создать методы для атаки, применения способностей и защиты.
	Добавить логику очередности ходов (например, по инициативе).

	5)Прокачка системы
	Разработать систему расчета опыта (XP) и уровней.
	Создать интерфейс для отображения уровня и прогресса в прокачке.
	Реализовать возможность улучшения характеристик и открытия новых способностей.

	6)AI для мобов
	Определить поведение нейтральных мобов: агрессивное, мирное.
	Реализовать логику спавна мобов и их взаимодействие с игроками.

	7)Интерфейс пользователя
	Создать меню выбора класса, экран загрузки и боевой интерфейс.
	Реализовать инвентарь для предметов и способности.

	8)Тестирование
	Провести тестирование механик игры, исправление багов и оптимизация производительности.
	Провести тестирование с игроками для получения обратной связи.
